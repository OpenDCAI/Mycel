### Why Mycel?

Existing agent frameworks focus on *building* agents. Mycel focuses on *running* them in production:

- **Middleware Pipeline** — Unified tool injection, validation, security, and observability
- **Sandbox Isolation** — Docker / E2B / cloud with automatic state management
- **Configuration-Driven** — Control all capabilities via JSON, no code changes
- **Production Governance** — Security controls, audit logging, cost tracking
